Here you will learn how ticketh works.

Requirements:
  • Chrome or Firefox browser.
  • Metamask
Rules:
  • Every player can buy more than one ticket. (buy more tickets if you are not the only player)
  • Every player can participate in more than one lottery.
  • Lottery starts when first ticket is bought for the round.
  • The lottery duration is current block number when starting + duration blocks.
  • Once the current block number reaches the lottery block number only one more ticket can be submitted.
  • When the last ticket is submitted the lottery ends and a winner is selected.
  • Winner is determined by a random number and prize is received within seconds.
  • Prize is calculated when there is more than one participant. 10% tax is for the lottery owner.
  • Displayed prize includes the last entry.
